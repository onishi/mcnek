# 主要導線の手動確認手順

リリース前や大きな変更の後に、以下を手動で確認する。`npm run dev` でローカルサーバーを起動してから行う。

## 一覧ページ（`/`）

- [ ] 道の駅が一覧表示され、件数が表示される
- [ ] 駅名で検索すると該当する駅だけに絞り込まれる
- [ ] 都道府県フィルター・地方フィルターで絞り込みができる
- [ ] 該当 0 件のときに「該当する道の駅が見つかりません」と表示される
- [ ] ページングで次のページ・前のページに移動できる
- [ ] 「行った道の駅だけ表示」をチェックすると訪問済みの駅だけになる
- [ ] 駅名をクリックすると詳細ページに移動する
- [ ] 「行った道の駅を見る」リンクから `/visited` に移動できる
- [ ] スマートフォン幅（375px 程度）でレイアウトが崩れない

## 詳細ページ（`/stations/:id`）

- [ ] 駅名・所在地・登録年月が表示される
- [ ] 緯度経度が未登録の駅で「未登録」と表示される
- [ ] 国土交通省・連絡会の参照リンクが正しい URL に飛ぶ
- [ ] 連絡会の詳細 URL が未登録の駅でその旨が表示される
- [ ] 「行った」ボタンを押すと「行った済み」に切り替わり、リロードしても保持される
- [ ] 存在しない駅 ID で「指定された道の駅が見つかりません」と表示される
- [ ] 「一覧に戻る」で一覧ページに戻れる

## 行った道の駅ページ（`/visited`）

- [ ] 訪問済みの駅だけが一覧表示される
- [ ] 都道府県別の訪問数（訪問済み / 総数）が表示される
- [ ] 全体の訪問数と達成率が表示される
- [ ] 訪問済みの駅が 0 件のとき、0 件と空の一覧表示になる

## データ照合ページ（`/data-check`）

- [ ] データ取得状況（取得日時・件数）が表示される
- [ ] 前回データとの差分が表示される（差分がない場合はその旨が表示される）
- [ ] 都道府県ごとに国土交通省データと連絡会データの一致状況が表示される
- [ ] 未一致の駅がある場合、手動紐付け候補をチェックできる
- [ ] 一度チェックした候補を再度外せる
- [ ] 「手動紐付けをJSONでコピー」でクリップボードにコピーできる

## オフライン対応（PWA）

- [ ] 一覧ページを開いた状態でネットワークを切断しても表示が崩れない
- [ ] オフライン時にデータが古い可能性がある旨のバナーが表示される
- [ ] ホーム画面に追加できる（モバイル端末で確認）

## 共通

- [ ] キーボードのみ（Tab / Enter）で主要な操作ができる
- [ ] ブラウザの戻る・進むで正しい画面に戻る
